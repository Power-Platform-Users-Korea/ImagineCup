## 📖 프로젝트 소개

이 프로젝트는 **노동자의 정당한 실업급여 권리를 보호**하기 위한 AI 상담 도우미입니다. 최근 쿠팡, 런던베이글뮤지엄 등에서 발생한 부당한 자진퇴사 강요, 불법 해고 등으로부터 노동자를 보호하는 24/7 AI 법률 상담 서비스입니다.

**주요 기능:**
- 🚨 긴급 상황 즉시 대응 (사직서 강요, 부당 압박)
- ⚖️ 부당한 상황 식별 및 경고
- 📝 증거 수집 방법 안내 (녹음, 문서, 증인)
- 🔗 무료 법률 지원 기관 연계
- 💪 실업급여 수급 자격 안내
- 📚 노동법 기본 지식 제공

---

## 🚨 왜 이 프로젝트가 필요한가

### 최근 한국의 현실

**쿠팡 사태 (2024)**:
- 수천 명의 노동자들에게 "자진퇴사" 강요
- 실업급여 수급 자격 박탈
- 노동자들은 자신의 권리를 몰랐음

**런던베이글뮤지엄 사건**:
- 정당한 해고 절차 없이 실업급여 회피
- 노동자들의 권리 침해

**전국 수많은 중소기업**:
- 복잡한 노동법 악용
- "합의 퇴사", "권고사직" 형태로 위장
- 노동자들의 정당한 권리 박탈

### 정보 격차의 문제

| 사업주 측 | 노동자 측 |
|----------|----------|
| ✅ 전문 노무사 자문 | ❌ 정보 없음 |
| ✅ 법률 지식 풍부 | ❌ 복잡한 제도 이해 어려움 |
| ✅ 전략적 대응 | ❌ 당황하여 즉각 대응 |
| ✅ 시간과 자원 | ❌ 긴급 상황 |

**결과**: 정당한 권리를 가진 노동자들이 실업급여를 받지 못함

## 💡 해결책: AI 권리 지킴이

### 핵심 기능

#### 1. 🚨 긴급 상황 즉시 대응

```
상황: "상사가 지금 당장 사직서 쓰라고 합니다"

AI 응답:
🚨 절대 서명하지 마세요!

지금 즉시:
1. 스마트폰 녹음 시작 (본인 참여 대화는 합법)
2. "시간을 주십시오"라고 말하고 자리 이탈
3. 고용노동부 1350 전화
4. 받은 문서 모두 사진 촬영

이것은 불법적인 실업급여 박탈 시도입니다.
```

#### 2. ⚖️ 부당한 상황 식별

**AI가 즉시 경고하는 위험 신호**:
- "자진 퇴사하면 추천서 써줄게"
- "합의 퇴사로 하면 퇴직금 더 줄게"
- "여기 사인만 하면 끝이야"
- "다른 직원들도 다 이렇게 했어"
- "법적 문제 없으니 걱정 마"

→ **즉시 전문가 상담 필요!**

#### 3. 📝 증거 수집 가이드

```
증거 수집 체크리스트:

□ 녹음
  - 스마트폰 녹음 앱 사용
  - 본인이 참여한 대화는 합법
  - 날짜, 시간 메모

□ 문서
  - 급여명세서 전체
  - 근로계약서
  - 이메일, 메시지 백업
  - 회사 내부 문서

□ 증인
  - 동료 연락처 확보
  - 같은 상황 겪은 사람 확인

□ 타임라인
  - 사건 경과 시간순 기록
  - 누가, 언제, 어디서, 무엇을
```

**대한법률구조공단**:

**한국노총/민주노총**:
- 한국노총: 02-6277-8000
- 민주노총: 02-2635-8989

#### 5. 💪 실업급여 수급 자격 안내

**기본 자격 (2024년 기준)**:
- 퇴사 전 18개월 중 180일 이상 근무
- 비자발적 이직:
  - ✅ 해고 (부당해고 포함)
  - ✅ 권고사직
  - ✅ 계약 만료
  - ✅ 회사 귀책사유 (임금 체불, 근로조건 위반)
  - ❌ 단순 자진퇴사

**수급액**:
- 퇴직 전 평균임금의 60%
- 최소: 하루 63,104원
- 최대: 하루 66,000원
- 기간: 120일 ~ 270일

## 핵심 역할

노동자들이 부당한 대우를 받거나, 실업급여 권리를 침해당하지 않도록 돕습니다.

## 주요 지원 분야

1. **실업급여 수급 자격 안내**
   - 자발적 퇴사 vs 권고사직 vs 해고 차이
   - 수급 요건 확인
   - 수급액 계산

2. **부당한 상황 식별**
   - "자진퇴사 유도" 수법 인식
   - "합의 퇴사" 압박 대응
   - 불법 해고 판단

3. **증거 수집 안내**
   - 녹음 방법
   - 문서 보관
   - 증인 확보

4. **대응 절차 안내**
   - 고용센터 신고
   - 노동청 진정
   - 무료 법률 지원

## 응답 원칙

### 1. 명확하고 단호하게
❌ "아마도...", "할 수도..."
✅ "이것은 불법입니다", "당신은 권리가 있습니다"

### 2. 즉각적인 행동 안내
지금 당장 무엇을 해야 하는지 명확히 제시

### 3. 법률 용어 쉽게 설명
- "비자발적 이직" → "회사가 당신을 그만두게 만든 것"
- "권고사직" → "회사가 그만두라고 권유 (실업급여 가능)"

### 4. 위험 신호 즉시 경고

다음 상황은 부당한 실업급여 박탈 시도:
- "자진 퇴사하면 추천서 써줄게"
- "합의 퇴사로 하면 퇴직금 더 줄게"
- "여기 사인만 하면 끝"

→ 절대 서명 금지! 즉시 전문가 상담!

# 실업급여 및 노동법 기본 정보

## 실업급여 수급 자격 (2024년 기준)

### 기본 요건
1. **피보험 기간**: 퇴사 전 18개월 중 180일 이상 근무
2. **비자발적 이직**: 본인 의사와 무관하게 퇴사

### 비자발적 이직 인정 사유
- ✅ 해고 (정당/부당 모두 포함)
- ✅ 권고사직
- ✅ 계약 기간 만료
- ✅ 회사 귀책사유:
  - 임금 체불 (2개월 이상)

## 수급액 계산

```
일일 수급액 = (퇴직 전 3개월 평균 임금) × 60%
```

- **최소**: 하루 63,104원
- **최대**: 하루 66,000원

## 신청 방법

### 2단계: 고용센터 방문
- 거주지 관할 고용센터
- 구직 등록 확인증 제출

### 3단계: 서류 제출
필요 서류:

- 통장 사본

### 4단계: 실업 인정
- 매 2주 또는 4주마다 고용센터 방문
- 구직 활동 증빙
- 실업 상태 확인

## 부당한 수법 식별

### 수법 1: 자진퇴사 유도

**회사 말**:
- "사직서 내면 조용히 보내줄게"
- "안 그러면 징계 처리할 거야"
- "다른 사람들도 다 이렇게 했어"

**대응**:
```
절대 사직서 쓰지 마세요!

"사직할 의사가 없습니다. 
만약 회사가 저를 보내고 싶으시다면 
정당한 절차에 따라 해고 통지를 해주십시오."
```

### 수법 2: 합의 퇴사 압박

**회사 말**:
- "합의하면 퇴직금 더 줄게"
- "소송하면 너만 손해야"
- "빨리 결정해, 이 기회 놓치면 끝이야"

**대응**:
```
즉시 서명하지 마세요!

"변호사와 상의하고 답변 드리겠습니다"

그리고:
1. 문서 사본 요청
2. 1350 전화 상담
3. 무료 노무사 상담 (132)
```

### 수법 3: 허위 사유로 징계 해고

**회사 행동**:
- 근거 없는 이유로 징계
- "너 업무 태만이야"
- "규정 위반했어"

**대응**:
```
1. 해고 통지서 서면 요구
2. 해고 사유 구체적으로 확인
3. 증거 자료 확보
4. 부당해고 구제 신청 (3개월 이내)
```

- 법적 증거로 인정

📱 녹음 방법:
- 스마트폰 기본 녹음 앱

- 날짜, 시간 메모

### 문서
```

```

## 대응 절차

### 1. 내용증명 발송

**목적**: 공식 기록 남기기

**내용**:
- 사직서 철회 (강요된 경우)
- 부당 처우 항의
- 임금 체불 요구

**방법**:
- 우체국 방문
- 내용증명 우편 발송
- 수령 확인

### 5단계: 백엔드 코드 수정

`src/api/app.py` 파일을 광명시 도서관과 동일한 구조로 수정:

```python
import os
from pathlib import Path
from openai import AzureOpenAI
from flask import Flask, request, jsonify
import logging

app = Flask(__name__)
logging.basicConfig(level=logging.INFO)

client = AzureOpenAI(
    api_key=os.getenv("AZURE_OPENAI_API_KEY"),
    api_version="2024-02-15-preview",
    azure_endpoint=os.getenv("AZURE_OPENAI_ENDPOINT")
)

def load_custom_prompts():
    """커스텀 시스템 프롬프트와 노동법 데이터 로드"""
    try:
        project_root = Path(__file__).parent.parent.parent
        
        prompt_path = project_root / "prompts" / "unemployment_assistant.md"
        with open(prompt_path, "r", encoding="utf-8") as f:
            system_prompt = f.read()
        
        data_path = project_root / "data" / "labor_law_info.md"
        with open(data_path, "r", encoding="utf-8") as f:
            labor_law_data = f.read()
        
        full_prompt = f"{system_prompt}\n\n# 노동법 정보\n\n{labor_law_data}"
        
        logging.info("✅ 실업급여 권리 지킴이 프롬프트 로드 성공")
        return full_prompt
    
    except Exception as e:
        logging.error(f"❌ 프롬프트 로드 실패: {e}")
        return "당신은 노동자의 권리를 돕는 AI 상담사입니다."

SYSTEM_PROMPT = load_custom_prompts()

@app.route("/chat", methods=["POST"])
def chat():
    """실업급여 권리 상담"""
    try:
        user_message = request.json.get("message", "")
        
        if not user_message:
            return jsonify({"error": "메시지가 제공되지 않았습니다"}), 400
        
        logging.info(f"📩 상담 요청: {user_message[:50]}...")
        
        response = client.chat.completions.create(
            model=os.getenv("AZURE_OPENAI_DEPLOYMENT_NAME", "gpt-4"),
            messages=[
                {"role": "system", "content": SYSTEM_PROMPT},
                {"role": "user", "content": user_message}
            ],
            temperature=0.7,
            max_tokens=1000
        )
        
        assistant_reply = response.choices[0].message.content
        logging.info(f"✅ 상담 응답 생성")
        
        return jsonify({
            "response": assistant_reply,
            "model": response.model,
            "usage": {
                "prompt_tokens": response.usage.prompt_tokens,
                "completion_tokens": response.usage.completion_tokens,
                "total_tokens": response.usage.total_tokens
            }
        })
    
    except Exception as e:
        logging.error(f"❌ 오류: {e}")
        return jsonify({
            "error": "죄송합니다. 오류가 발생했습니다.",
            "emergency_contact": "고용노동부 1350으로 즉시 연락하세요"
        }), 500

@app.route("/health", methods=["GET"])
def health():
    return jsonify({"status": "healthy", "service": "실업급여 권리 지킴이"})

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=8000, debug=False)
```
